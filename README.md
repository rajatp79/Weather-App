# 🌤️Weather App

A premium, modern, and highly interactive Weather Application built with Vanilla Web Technologies. It provides real-time weather data, dynamic cinematic backgrounds based on weather conditions, and a sleek frosted-glass (Glassmorphism) UI.

**[🔴 Live Demo Here](https://github.com/rajatp79/Weather-App.git/)**

## ✨ Key Features

* **Real-Time Weather Data:** Fetches accurate temperature, humidity, wind speed, and weather conditions using the OpenWeatherMap API.
* **Cinematic Dynamic UI:** The application's background seamlessly transitions to match the current weather conditions (Clear, Rain, Clouds, Snow, etc.) with a premium frosted-glass container.
* **Smart Auto-Suggest Search:** Integrated Geocoding API to provide real-time city name suggestions in a dropdown as the user types.
* **Geolocation Support (📍):** Users can fetch weather details for their exact current location with a single click using the browser's Geolocation API.
* **Persistent Recent Searches:** Utilizes `localStorage` to save and display the user's recent successful searches, auto-loading the last searched city upon refreshing the app.
* **Fully Responsive:** Optimized for both desktop and mobile screens.

## 🛠️ Tech Stack

* **Frontend:** HTML5, CSS3 (Glassmorphism Design, CSS Variables, Animations)
* **Logic:** Vanilla JavaScript (ES6+, Async/Await, DOM Manipulation)
* **APIs Used:** * [OpenWeatherMap Current Weather API](https://openweathermap.org/current)
  * [OpenWeatherMap Geocoding API](https://openweathermap.org/api/geocoding-api)
* **Storage:** Browser Local Storage
